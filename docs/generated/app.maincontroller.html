class app.maincontroller.MainController(view, system)[source]¶

The controller is responsible for the control flow of the application. It acts as the glue of the application by connecting the GUI to the Anser (EMT) system. It delegates UI events to the appropriate Anser (EMT) function, vice versa.

Parameters:
  • view – mainwindow instance (representing the UI)
  • system – qtanser instance (representing the Anser System)
