Main Files¶

Main Executable

application Runs the application

Interfaces

qtanser Qt Interface for the Anser system
mainwindow The primary window and user interface for the application
maincontroller Connects the UI to the Anser system
calibrationthread Qt interface for performing a calibration

GUI¶

Graphs¶

gridgraph Represents the field generator board used during calibration
fftgraph Shows the frequencies emitted by the field generator in real time
positiongraph Displays sensor positions in real time

Panels¶

calibrationpanel Starts calibration and creates/removes sensors
igtpanel Creates an OpenIGTLink server
systempanel Starts tracking and displays system info

Tabs¶

systemtab Starts tracking and displays system info
igttab 3D Visualisation and OpenIGTLink
calibrationtab Starts calibration and creates/removes sensors
configeditortab System settings editor

Specialised Widgets¶

tabwidget The container of all tabs
loggerwidget Logs debug information
