Welcome to Qt-Anser's documentation!
==================================

**Qt-Anser** is the graphical user interface (GUI) for the  `Anser
<https://github.com/StephenHinds/AnserEMT-Homepage/>`_ electromagnetic tracking system based on the `Python-Anser
<https://github.com/StephenHinds/AnserEMT-Homepage/>`_ library.
It is built using the PyQt platform and Pyqtgraph a graphics tool for 2D/3D (OpenGL) visualisation.
